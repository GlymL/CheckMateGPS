# IMPORTANTE

Extensiones y paquetes importantes para usar VSCode
Primero --> Git Extension Pack. Este paquete sirve para conectarte a GITHUB, sin este paquete no se puede hacer nada con el repositorio.
Segundo --> Java y Extension Pack for Java.
Tercero --> Spring Boot Extension Pack.

Para que java funcione, hay que descargar esto e instalarlo.

https://code.visualstudio.com/docs/languages/java

Una vez se tenga esto, se puede ejecutar Java en VSCode, sin esto, el proyecto no se podrá ejecutar. También hay que utilizar Spring Boot para que sea una aplicación Web. 
Una vez el proyecto complie, no habrá un output como de costumbre. Hay que abrir localhost:8080 para ver la aplicación.

Para que las cosas sean persistentes, deberíamos utilizar una Base de Datos. Como solo sabemos SQL, tendremos que usar una base de datos SQL, yo recomiendo MySQL. Pero cualquiera debería funcionar.

He hecho un main de prueba para ver como funciona, porbad a ver si os compila, para poder empezar a trabajar en todo esto.
